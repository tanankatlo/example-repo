# hello_world

## Description

This is a simple Python project I made while learning the basics of Git and version control. The program prints a message to the terminal.

At first, it printed "Hello World!", but I later changed it to "Git is awesome!". This helped me see how Git keeps track of changes and saves different versions of the same file.

Working on this project also helped me practice creating a repository, making changes, staging files, and committing them with Git.

## How to Run

Make sure you have Python installed on your computer.

Open your terminal, go to the project folder, and run this command:

```bash
python hello_world.py
```

You should see this output:

```text
Git is awesome!
```
